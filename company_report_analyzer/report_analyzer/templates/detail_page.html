Company Report Analyzer

Company Report Analyzer is an application that processes a data file in a few clicks and produces the requested reports. To demonstrate the capabilities of this application, I used a test data file and created 8 possible test reports. The test result is of no value at all. But in the real world, the value and potential of this application is limitless. Therefore, I urge you not to look at this application

through the prism of a test task...

In the real world (if the application is modified and customized to the needs of a specific company), this application will be able to process ANY file with company data.
